# Worship Set Curation Knowledge Base

## Worship Flow Structure (Korean Church Context)

Korean Protestant worship services typically follow this flow pattern:

1. **Opening / 시작 (환영)** — Upbeat, congregational engagement. Sets energy and expectation.
2. **Praise / 찬양** — High-energy, celebratory songs. Key: major keys, faster tempos (120–140 BPM). 1–2 songs.
3. **Worship / 경배** — Transitional, moving from outward praise to inward devotion. Medium tempo (90–110 BPM). 1–2 songs.
4. **Encounter / 만남 (깊은 경배)** — Intimate, reflective. Slower tempo (60–90 BPM), minor keys or gentle major keys. The congregation meets God. 1–2 songs.
5. **Response / 응답** — After the sermon or encounter, a song of commitment or prayer. Often a hymn or well-known CCM. 1 song.
6. **Closing / 마무리 (축복)** — Sends the congregation out. Can be upbeat or a benediction. 1 song.

## Key Transition Rules

### Preferred Transitions (Smooth)
- **Same key**: C → C (key change within song)
- **Relative minor/major**: C major → A minor, G major → E minor
- **Fourth up**: C → F, G → C (very natural resolution feel)
- **Fifth up**: C → G, D → A (bright, lifting transition)
- **Half step up**: C → Db (dramatic modulation, use sparingly)
- **Whole step up**: C → D (energizing modulation)

### Transitions to Avoid
- **Tritone jump**: C → F# (jarring, dissonant)
- **Minor third down**: C → A (awkward unless relative)
- **Random jumps**: More than 3 semitones without harmonic relationship

### Best Practices
- Between praise songs: fourths/fifths are ideal
- Transitioning to worship: move toward relative minor or step down
- For encounter: settle into a home key, minimize transitions
- Use capo-friendly keys for guitar-led worship: G, C, D, E, A, Em, Am

## Korean Church Worship Context

### Song Categories
- **Korean CCM (찬양)**: Original Korean worship songs (e.g., 마커스워십, 제이어스, 아이자야식스티원)
- **Translated Western Worship (번역 찬양)**: Korean translations of Hillsong, Bethel, Elevation, etc.
- **Korean Hymns (찬송가)**: Traditional hymns from the Korean hymnal (찬송가)
- **Contemporary Hymns**: Modern arrangements of traditional hymns

### Liturgical Seasons (절기)
- **대림절 (Advent)**: Hope, expectation — "오소서" themed songs
- **성탄절 (Christmas)**: Joy, celebration — carols and nativity songs
- **사순절 (Lent)**: Repentance, reflection — minor keys, introspective
- **부활절 (Easter)**: Victory, resurrection — triumphant, major keys
- **성령강림절 (Pentecost)**: Holy Spirit — fire, wind imagery
- **추수감사절 (Thanksgiving)**: Gratitude — harvest and blessing themes

### Congregation Considerations
- Balance familiar songs (70%) with newer songs (30%) for engagement
- Consider multigenerational congregations: mix CCM with hymns
- For youth services: lean toward contemporary CCM and translated worship
- For early morning services (새벽예배): more reflective, hymn-focused

## Song Selection Heuristics

### Theme Matching
- Match song lyrics to the sermon scripture or theme
- Group songs by theological topic: grace, holiness, praise, surrender, mission
- Consider the emotional journey: start broad (God's greatness) → personal (my response)

### Tempo Grouping
- Fast songs (120+ BPM) → Praise section
- Medium songs (90–119 BPM) → Worship/transition section
- Slow songs (60–89 BPM) → Encounter/response section

### Duration Planning
- Average song duration: 4–5 minutes (including intro/outro)
- Transition between songs: 30 seconds – 1 minute
- Prayer/scripture reading moments: 1–3 minutes
- For a 25-minute worship block: 5–6 songs
- For a 40-minute worship block: 7–9 songs
- For a 15-minute worship block: 3–4 songs

### Tag-Based Selection
- Use song tags (e.g., "praise", "intimate", "declaration") to match flow positions
- Use topic tags to align with sermon themes
- Consider language tags for bilingual services

## Output Quality Guidelines

- Every song must have a clear rationale for its position
- Transition notes should explain the musical and spiritual connection
- Key suggestions should account for vocalist range (most Korean worship in D–A range)
- Prefer songs from the user's actual song library — never suggest songs not in the database
