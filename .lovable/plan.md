

# 다른 커뮤니티 세트 배너 스타일 개선

## 현재 상태 vs 개선 후

```text
현재:
┌──────────────────────────────────────────────┐
│ 🔒 다른 예배공동체의 공개된 예배세트 (읽기 전용) │
│    👁 조회수: 12회                              │
└──────────────────────────────────────────────┘
스타일: bg-accent/10, border-accent/30

개선 후:
┌─────────────────────────────────────────────────────────────────┐
│ ┌─────┐                                                         │
│ │ 🌐 │  다른 예배공동체의 워십세트  [읽기 전용]                 │
│ └─────┘  이 세트는 참고용으로만 열람할 수 있습니다              │
│                                                                 │
│          ┌────────────────┐                                     │
│          │  👁 조회 12회   │                                     │
│          └────────────────┘                                     │
└─────────────────────────────────────────────────────────────────┘
스타일: 그라디언트 배경 + 원형 아이콘 + 계층적 텍스트
```

## 수정 파일

### 1. src/pages/BandView.tsx

**변경 내용:**
- Lock → Globe 아이콘으로 변경
- 그라디언트 배경과 개선된 레이아웃 적용
- 아이콘에 원형 배경 추가
- 제목 + 뱃지 + 서브텍스트 구조화
- 조회수 pill 형태 뱃지

**Import 추가:**
```tsx
import { Globe } from "lucide-react";
```

### 2. src/lib/translations.ts

**추가할 번역 키 (songUsage 섹션):**

| 키 | 영어 | 한국어 |
|----|------|--------|

| `readOnly` | Read-only | 읽기 전용 |

## 디자인 개선 포인트

| 요소 | 기존 | 개선 |
|------|------|------|
| 배경 | 단색 `bg-accent/10` | 그라디언트 `from-primary/10 via-primary/5 to-accent/10` |
| 아이콘 | Lock (제한적 느낌) | Globe (열린 공유 느낌) + 원형 배경 |
| 레이아웃 | 수평 단순 정렬 | 아이콘 분리 + 계층적 텍스트 |
| 텍스트 | 단일 라인 | 제목 + [읽기 전용] 뱃지 + 부제목 |
| 조회수 | 일반 텍스트 | pill 형태 뱃지 (bg-background/80) |
| 테두리 | accent 색상 | primary 색상 (브랜드 통일) |
