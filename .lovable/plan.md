

# 모바일 빌딩 높이 확대 + 도로 하단 고정

## 현재 상태
- 빌딩 본체: `h-[38vh]` → 빌딩이 작고 하단에 빈 하늘 배경 공간이 많음
- 도로가 빌딩 바로 아래에 위치하지만, 그 아래에 빈 하늘 영역이 남아있음

## 변경 사항 (`StudioSidePanel.tsx`)

### 1. 빌딩 본체 높이 확대
- Line 695: `h-[38vh]` → `h-[50vh]`로 증가
- 빌딩 층이 더 많은 공간을 차지하여 빈 배경 줄어듦

### 2. 도로를 패널 맨 하단에 고정
- Line 683의 빌딩 래퍼(`relative z-10 flex flex-col flex-1 min-h-0`)에서 `flex-1`이 이미 남은 공간을 채우지만, 내부 구조가 `shrink-0` 요소들과 `h-[38vh]` 고정 높이로 인해 빈 공간이 생김
- 빌딩 래퍼를 `flex-1 justify-end`로 변경하여 컨텐츠를 하단으로 밀어냄 → 도로가 자연스럽게 패널 최하단에 위치

### 3. 하늘 spacer 축소
- Line 680: 모바일 spacer `h-5` → `h-3`으로 줄여 상단 빈 공간 최소화

## 결과
- 빌딩이 화면의 더 많은 부분을 차지
- 도로가 시트 맨 아래에 붙음
- 빈 하늘 배경 공간 최소화
- 스크롤 동작은 기존 그대로 유지 (층만 스크롤, 상가/도로 고정)
