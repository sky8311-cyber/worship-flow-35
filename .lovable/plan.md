

# 팀멤버 화면에 공동체 검색 안내 추가

## 현재 상태

```text
┌─────────────────────────────────────────────────────────┐
│  예배팀 멤버로 참여                                      │
│  "기존 팀에 초대받아 참여하기"  ← 수동적인 안내만 있음     │
└─────────────────────────────────────────────────────────┘
```

## 개선 내용

팀멤버 선택 시 **능동적으로 공동체를 찾을 수 있다**는 안내 추가:

```text
┌─────────────────────────────────────────────────────────┐
│  예배팀 멤버로 참여                                      │
│  "초대를 받거나, 공동체를 직접 찾아서 가입할 수 있어요"   │
└─────────────────────────────────────────────────────────┘
```

---

## 수정 파일

**파일**: `src/components/onboarding/RoleSelectionDialog.tsx`

**변경 위치**: Line 135-139 (팀멤버 옵션 설명 부분)

```tsx
// 변경 전
<p className="text-sm text-muted-foreground mt-1">
  {language === "ko" 
    ? "기존 팀에 초대받아 참여하기" 
    : "Join an existing team through invitation"}
</p>

// 변경 후
<p className="text-sm text-muted-foreground mt-1">
  {language === "ko" 
    ? "초대를 받거나, 공동체를 직접 찾아서 가입" 
    : "Get invited or search for a community to join"}
</p>
```

---

## 예상 결과

사용자가 팀멤버를 선택할 때:
- 초대받는 것만 가능한 게 아니라
- **직접 공동체를 검색해서 가입 요청**할 수 있다는 것을 인지

이렇게 하면 `/community/search` 기능을 더 많이 활용하게 되어 자연스러운 커뮤니티 참여 증가 기대
