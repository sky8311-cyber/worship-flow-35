

# 히어로 애니메이션 재구성

## 레이아웃 개념

```text
┌──────────────────────────────────────┐
│                                      │
│        ╭──────────╮                  │
│       ╱   아치심볼  ╲                 │
│      │              │                │
│      │              │                │
│ "예배를"         "삶의 흐름으로"       │  ← 각각 아치 다리 끝 위치에 배치
│                                      │
│         "연결합니다"                   │
│                                      │
│  ── Phase 6 후 ──                    │
│    "예배를 삶의 흐름으로"              │  ← 양쪽 텍스트가 가운데로 모여 한 줄
│         "연결합니다"                   │
│                                      │
│       [ 내 공간 만들기 ]              │
└──────────────────────────────────────┘
```

## 애니메이션 순서 (6단계)

| Phase | 내용 | 타이밍 |
|-------|------|--------|
| 1 | "예배를" — 왼쪽 아치 다리 아래, 타이프라이터 | 0.4s |
| 2 | 아치 드로잉 시작 (outer arch) | ~1.0s |
| 3 | 아치 드로잉 완료 (inner arch + 별 stamp) | ~3.0s (자동) |
| 4 | "삶의 흐름으로" — 오른쪽 아치 다리 끝에서 시작, 타이프라이터 | 아치 완료 후 |
| 5 | "연결합니다" — 중앙 아래, 타이프라이터 + 커서 | Phase 4 완료 후 |
| 6 | "예배를" + "삶의 흐름으로" 가 `motion`으로 x 이동하여 중앙에서 한 줄로 합침 → CTA 페이드인 | Phase 5 완료 후 |

## 수정 파일

### `AtelierHero.tsx` — 전면 재작성

**레이아웃**:
- 아치 + 텍스트를 `relative` 컨테이너로 감싸기
- "예배를": 아치 왼쪽 다리 아래 `absolute` 배치 (left 정렬)
- "삶의 흐름으로": 아치 오른쪽 다리 끝에서 시작 `absolute` 배치 (right 정렬)
- Phase 6에서 두 텍스트의 `x` 좌표를 `motion.animate`로 이동시켜 중앙 합침

**타이프라이터 구현**:
- "예배를", "삶의 흐름으로", "연결합니다" 각각에 글자 단위 타이프라이터 state
- 각 텍스트에 `charIndex` state → 한 글자씩 증가 → 완료 시 다음 phase 트리거

**Phase state machine**:
- `phase`: 0→6 단계
- Phase 1 완료(예배를 타이핑 끝) → Phase 2 (아치 시작)
- `onArchComplete` 콜백으로 Phase 3→4 전환
- Phase 4 완료(삶의 흐름으로 타이핑 끝) → Phase 5
- Phase 5 완료(연결합니다 타이핑 끝) → Phase 6 (합침 + CTA)

**Phase 6 합침 효과**:
- "예배를"와 "삶의 흐름으로"를 `motion.span`으로 감싸고, phase 6에서 `x` 값을 animate하여 중앙으로 이동
- `absolute` → 합침 후 한 줄 inline으로 전환

### `AtelierArchLogo.tsx` — onArchComplete 활용
- `onArchComplete`가 inner arch 완료 시 호출되도록 유지 (현재 이미 구현됨)
- Hero에서 이 콜백으로 phase 전환
