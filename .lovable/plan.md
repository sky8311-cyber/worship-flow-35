

# "예배를" 위치 수정

## 현재 문제
"예배를"가 아치 **위**에 위치해 있음. 원래 의도는 아치 **아래**, "삶으로" 왼편에 배치하여 한 줄로 읽히는 것: **"예배를 삶으로"**

## 변경 내용 (`AtelierHero.tsx`)

- "예배를" `motion.span`을 아치 위에서 제거
- 아치 아래 텍스트 블록에서 "예배를"과 "삶으로"를 한 줄(`inline`)로 배치:
  ```
       [아치 심볼]
    예배를 삶으로
     연결합니다.
  ```
- "예배를"는 Phase 1에서 페이드인, "삶으로"는 Phase 3에서 타이프라이터 — 같은 줄 안에서 순차 등장
- 첫 번째 줄: `flex-row items-baseline gap-1`로 인라인 배치
